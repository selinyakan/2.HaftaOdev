JavaScript ile Neler Yapabilirsiniz?

Javascript Netscape firması tarafından C tabanında geliştirilmiş bir script dilidir. Script dillerinin özelliği derlenebilir bir dosya oluşturmaz. JS uzantılı dosyalarla sunucu üzerinden çalışır. Javascript web sayfalarında client tarafından çalıştığı için hızlıdır.

JavaScript Özellikleri

  1. Kullanıcı taraflı çalışan bir dildir. (client)
  2. Javascript kodlarıyla bilgisayardan herhangi bir dosya okunamaz,yazılamaz ve dosya oluşturulamaz. Çerezler bu durumun dışındadır.
  3. Html sayfaların bir parçasıdır. Tek başlarına bir anlam ifade etmezler.
  4. Tarayıcılar tarafından yorumlanır. Bu yine kullanıcı taraflılığıyla ilgilidir.
  5. Html içine gömüldüğü gibi .js uzantılı ek sayfa olarakta kullanılabilir.
  6. Herhangi bir metin editörüyle geliştirilebilir.
  7. JavaScript, HTML ve Web'in programlama dilidir.
  8. JavaScript öğrenmesi kolaydır.
  9. JavaScript ile HTML İçeriğini Değiştirebilirsiniz.
  10. JavaScript içinde metod adı verilen ve sizin hızlıca işlem yapmanızı sağlayan komutlar vardır. Bunlardan en çok kullanılanı getElementById() metodudur.

  11. Javascript yazarken hem çift tırnak (" ") hem de tek tırnak kullanabilirsiniz (' ')
  12. JavaScript ile HTML Özniteliklerini Değiştirebilirsiniz.
  13. JavaScript ile HTML Stillerini Değiştirebilirsiniz.
  14. JavaScript ile Bir HTML Elementini Gizleyebilirsiniz.
